A simple **flashcard app** built with **Jetpack Compose**, **Room database**, and **MVVM architecture**. Add, view, edit and delete flashcards with a clean UI and persistent storage.

---

## âœ¨ Features

- Add new flashcards (question & answer)
- View flashcards using `Previous` / `Next`
- Show / hide flashcard answers
- Edit flashcard
- Delete flashcards
- Persistent data using **Room**
- Modern **Jetpack Compose UI**
- **MVVM architecture**
- Navigation with **Navigation Compose**
